# 🏗️ TTB-Corp Website

Official website for **TTB-Corp** — a company specializing in **business strategy consulting and industrial steel structure construction**.

---

## 🚀 Project Overview

This project is a modern company landing page designed to showcase:

* 🏢 Company introduction
* 🏗️ Industrial steel structure solutions
* 📊 Business strategy consulting
* 🛠️ Featured construction projects
* 🤝 Partnership & contact section

---
